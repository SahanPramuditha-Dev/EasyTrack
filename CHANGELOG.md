# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2023-10-27

### Added
- Initial release.
- Added SQLite persistence for task storage.
- Implemented MVVM architecture with LiveData for reactive UI.
- User authentication (Sign-in/Sign-up) with session management.
- Modern Material 3 UI design.
- Profile customization with image upload support.
